A monolithic x86-64 kernel made using C and the GRUB bootloader.

# What can I do with this?
At the time of writing this, the kernel only has a VGA text mode library that allows you to print strings, no keyboard input just yet.
Feel free to use the code for whatever you want (or even contribute to it!), although I don't really recommend using my code since it sucks.

# Is this a GNU/Linux distribution?
No.

# How do I use this?
You can compile this kernel by running ```make``` with all the dependencies installed (which I'm too lazy to list), and then run ```make run``` to boot up the ISO using QEMU!
